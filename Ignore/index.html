Directive Name : Ignore

Link : Alpine.js Documentation

x-ignore merupakan directive akan akan membuatkan Alpine.js untuk tidak membaca code yang sepatutnya dia baca. Secara default, Alpine.js akan membaca semua DOM tree yang ada data x-init atau x-data. Tapi terkadang kita tidak mahu untuk Alpine baca. jadi kita gunakan directive ini untuk elak. Semak pada code dan browser untuk contoh
