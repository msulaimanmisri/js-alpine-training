Directive Name : Init

Link : Alpine.js Documentation

x-init Directive membenarkan kita untuk run the code sewaktu initialization berlaku. Dia macam start-up command untuk sesuatu benda. Dia akan jalan dulu sebelum benda lain berlaku. Macam kalau kita buka kedai, x-init ni macam kita buka pintu kedai dulu sebelum pelanggan masuk.
